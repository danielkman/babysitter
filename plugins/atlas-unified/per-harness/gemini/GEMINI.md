# Atlas (Gemini)

Atlas turns a stated need into a full system design by mining the Atlas
knowledge graph. Use the atlas skill and the `mcp__atlas__atlas_public_*` MCP
tools for all graph queries; delegate non-trivial runs to `babysitter:babysit`
with an atlas `.a5c` process.

The Atlas MCP server is wired natively for Gemini and defaults to
`https://atlas-staging.a5c.ai/api/mcp`, overridable via `ATLAS_MCP_URL`.
