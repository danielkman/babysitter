# @a5c-ai/genty-web-app

Browser-based genty agent control and monitoring application.

## Stack
- Vite + React 19 + React Router v6
- Shared components from `@a5c-ai/genty-ui`

## Development
```bash
npm run dev --workspace=@a5c-ai/genty-web-app
```

## Build
```bash
npm run build --workspace=@a5c-ai/genty-web-app
```
